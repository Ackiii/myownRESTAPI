Arnes Rest Api

Hi, this is my self written Rest API for demonstration purpose

Following "GET" Requests are supported, call to the Service via "/api/data/x" where x is one of the following parameters

Parameters for "GET"

  • prename
  • name
  • fullname
  • birthday
  • url
  • height
  • favanimal
  • obsession
  • post
  • all

Parameters for "POST" via "api/data/x" where x is a 10 Bit - Number

The number of Post will change with that request

  • any Number
